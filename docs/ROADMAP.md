# Development Roadmap: iTambayan

## Phase 1: Foundation (Weeks 1-2)
- [ ] Core Architecture Setup (Monolith with microservice hooks).
- [ ] Database Schema Implementation (PostgreSQL + Prisma).
- [ ] Authentication System (JWT + OAuth).
- [ ] Core UI Shell (Sidebar, Navbar, Feed Layout).

## Phase 2: Social Core (Weeks 3-5)
- [ ] Friend System (Requests, Suggestions, Mutuals).
- [ ] Profile Management (Photos, Bio, Experience).
- [ ] Feed Engine (Basic chronological + basic ranking).
- [ ] Reactions and Comments System.

## Phase 3: Real-Time & Media (Weeks 6-8)
- [ ] Messenger (Direct & Group Chat with Socket.io).
- [ ] Stories System (24h ephemeral content).
- [ ] Media Management (S3 Integration, Image/Video Processing).
- [ ] Real-time Notifications.

## Phase 4: Expansion (Weeks 9-12)
- [ ] Groups and Pages.
- [ ] Marketplace (Listings, Search, Categories).
- [ ] Reels (Short video feed).
- [ ] Events and RSVP System.

## Phase 5: Intelligence & Optimization (Weeks 13-16)
- [ ] Gemini AI Feed Recommendation Engine.
- [ ] AI Content Moderation.
- [ ] Performance Tuning (Redis, Sharding).
- [ ] Admin Dashboard and Analytics.

## Phase 6: Global Scale (Future)
- [ ] Multi-region Deployment.
- [ ] CI/CD Pipeline Maturity.
- [ ] Mobile App Development (React Native).
